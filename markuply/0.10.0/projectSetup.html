Project setup

Dependencies

To start using Markuply just add it as a dependency. Spring WebFlux is a transitive dependency so you don’t have to add yourself.

build.gradle.kts
dependencies {
    implementation("io.wttech.markuply:markuply-spring-boot-starter:0.10.0")
}

Maven

pom.xml
<dependencies>
  <dependency>
    <groupId>io.wttech.markuply</groupId>
    <artifactId>markuply-spring-boot-starter</artifactId>
    <version>0.10.0</version>
  </dependency>
</dependencies>
